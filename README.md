# Ecoyaan Checkout Flow

This is a sample frontend project for ecoyaan-checkout. It implements a responsive, multi-step checkout flow using React, Next.js (Pages Router), and Tailwind CSS.

## Features
- **Cart / Order Summary**: Displays products, calculates subtotals, and maps data from a mock API.
- **Shipping Address**: Form with client-side validation for address details.
- **Payment Method**: Simulated secure payment processing screen.
- **State Management**: Uses React Context API to manage and persist checkout state across steps.

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- React
- Next.js (Pages Router)
- Tailwind CSS
